# Map Explorer - Menus

_This page documents the menu bar in the Map Explorer. For the menu bar in the main window, see [Main Window - Menus](session-menus.md)._

## Menus

### Map

* <ins>Waypoints...</ins>: Opens the [Waypoints dialog](map-waypoints.md) to manage the waypoints for the current server.
* <ins>Settings...</ins>: Opens the [Map Settings dialog](map-settings.md) to manage mapping settings.
* <ins>Close</ins>: Closes the Map Explorer window.

### View

* <ins>Zoom In</ins>: Zooms in on the map view.
* <ins>Zoom Out</ins>: Zooms out on the map view.
* <ins>Always On Top</ins>: When checked, keeps the Map Explorer window visible above all other windows.

### Help

* <ins>Help</ins>: Shows a list of supported [commands](map-commands.md) above the command line.
